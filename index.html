Key Features

Why Resolume HUD?

Built for the demands of live performance, where every second counts and screen real estate is precious.

◎

Custom Widget Creator

Design your own overlay widgets tailored to your specific setup. Map any Resolume parameter to a visual indicator that stays visible on your screen at all times.

⟐

OSC-Driven Architecture

Built on Open Sound Control for reliable, low-latency communication with Resolume. Real-time parameter feedback with no perceptible delay.

◈

Show Mode Monitoring

Keep critical parameters visible even when Resolume is in Show Mode. Monitor colorize effects, opacity levels, and custom parameters without opening effect panels.

⬡

MIDI-Aware Feedback

See real-time visual feedback as you twist encoders and push faders. Color boxes, value readouts, and state indicators that respond instantly to your MIDI input.

◇

Lightweight Overlay

Runs as a transparent overlay on top of any application. Minimal CPU and GPU footprint — designed to run alongside demanding visual software without impacting performance.

⏣

Fully Customizable

Control widget size, position, color schemes, and display format. Build the heads-up display that matches your workflow and your aesthetic.
