v1.5 March 2026

IMPORTANT: If you used API widgets (not OSC) in v1.4, you'll need to re-create them once in the wizard. After that, they will automatically reconnect whenever Resolume restarts. This is a one-time step. OSC widgets are not affected.

API Connection Overhaul: Completely rebuilt API connection — parameter IDs now resolve fresh on every connection. Widgets automatically reconnect when Resolume restarts. Widgets show current values on connect. Works reliably regardless of start order. New Refresh API button in the status bar for manual reconnect.

New & Improved: Speed widget shows correct percentage via API. Clip transport parameters now visible in API browser. Boolean Indicator redesigned with large green/red status circle and ON/OFF overlay. Code signed with Microsoft Trusted Signing — no more SmartScreen warnings.

Under the Hood: Full security and code audit. Widget system refactored with shared base class. Update and What's New dialogs rebuilt. Config system hardened. Improved resource cleanup and thread safety.
